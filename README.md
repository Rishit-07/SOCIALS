# ⚡SOCIALS

**Make consistency a competition.**

MicroChallenge is a social productivity platform where users create short, focused challenges — like "read 10 pages daily" or "no sugar for 7 days" — and invite friends to compete together. Everyone checks in once a day, streaks are tracked automatically, and a live leaderboard keeps the motivation real. When the challenge ends, the top performer earns a reward.

# 🛠 Tech Stack

Frontend — React, Tailwind CSS

Backend — Node.js, Express

Database — MongoDB, Mongoose

Auth — JWT, bcryptjs

Realtime — Socket.io


# ✨ Features

Create and join challenges via invite code
Daily check-ins with optional notes
Streak tracking and leaderboard
JWT-based authentication
Reward system for top performers


🚀 Getting Started
bash# Install backend dependencies
cd server && npm install

# Install frontend dependencies
cd client && npm install

# Run backend
cd server && npm run dev

# Run frontend
cd client && npm start
